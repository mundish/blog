The idea here is to figure out the translation of a random sentence I've come up with:
_You're only mad because they stole the izakaya's tame monkey._

My initial effort to gather the pieces needed yielded this:

- <span lang="ja">
    <Furigana readings={['おこ']}　>怒</Furigana> る
  </span> - To get angry/mad
- <span lang="ja">
    <Furigana readings={['い', 'ざか', 'や']}>居酒屋</Furigana>
  </span> - Izakaya
- <span lang="ja">
    <Furigana readings={['ぬす']}>盗</Furigana> む
  </span> - To steal
- <span lang="ja">
    <Furigana readings={['おん','じゅん']}>温順</Furigana> (な)
  </span> - Tame (adj.)
- <span lang="ja">A(の)ために...</span> - Because of A ...

Which would result in something like:

<BlockQuote lang={'ja'}>
  <Furigana readings={['い', 'ざか', 'や']}>居酒屋</Furigana>の
  <Furigana readings={['おん', 'じゅん']}>温順</Furigana>
  なサルを
  <Furigana readings={['ぬす']}>盗</Furigana>
  んだために
  <Furigana readings={['おこ']}>怒</Furigana>
  られました。
</BlockQuote>

_However_, reaching out to a friend (native speaker), I got a few pointers:

- Tame has a lot of possible translations, but in this case, <span lang="ja">”飼いならす”</span> or <span lang="ja">“おとなしい”</span> sounds better.
- The <span lang="ja">に</span> particle after <span lang="ja">ため</span> can be omitted, making it a bit more conversational (like with a lot of particle uses).

So what I ended up with was the following:

<BlockQuote lang={'ja'}>
  <Furigana readings={['い', 'ざか', 'や']}>居酒屋</Furigana>
  のおとなしいサルを
  <Furigana readings={['ぬす']}>盗</Furigana>
  んだため、
  <Furigana readings={['おこ']}>怒</Furigana>
  られました。
</BlockQuote>

## Take aways:

Apart from the new vocabulary itself, what I learned was the concept of <span lang="ja">ため</span>:

<span lang="ja">ため</span> have other possible uses, but in this case the concept was that of a cause having an effect.

Both cause and effect can be either a noun or an action (verb).

Given the structure `cause` <span lang="ja">(の)ため(に/の)</span> `effect`, the following rules applies:

- _Noun_ <span lang="ja">のための</span> _noun_
- _Noun_ <span lang="ja">のため(に)</span> _verb_
- _Verb_ <span lang="ja">ための</span> _noun_
- _Verb_ <span lang="ja">ため(に)</span> _verb_
