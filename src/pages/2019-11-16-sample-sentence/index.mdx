import Furigana from '../../components/Furigana';

The idea here is to figure out the translation of a random sentence I've come up with:
_You're only mad because they stole the izakaya's tame monkey._

<br/>
My initial effort to gather the pieces needed yielded this:
<br/>
<Furigana characters={[{kanji:'怒',kana:'おこ'}]}　/>る - To get angry/mad
<br/>
<Furigana characters={[{kanji:'居',kana:'い'},{kanji:'酒',kana:'ざか'},{kanji:'屋',kana:'や'}]} /> - Izakaya
<br/>
<Furigana characters={[{kanji:'盗',kana:'ぬす'}]} />む - To steal
<br/>
<Furigana characters={[{kanji:'温', kana:'おん'},{kanji:'順', kana:'じゅん'}]}/>(な) - Tame (adj.)
<br/>
A(の)ために... - Because of A ...

<br/>
<br/>
Which would result in something like:
<br/>

<div className={'text-lg text-center'}>
    <Furigana characters={[{kanji:'居', kana:'い'},{kanji:'酒',kana:'ざか'},{kanji:'屋',kana:'や'}]} />
    の
    <Furigana characters={[{kanji:'温', kana: 'おん'},{kanji:'順',kana:'じゅん'}]} />
    なサルを
    <Furigana characters={[{kanji:'盗', kana:'ぬす'}]} />
    んだために
    <Furigana characters={[{kanji:'怒',kana:'おこ'}]}/>
    られました。
</div>
<br/>

_However_, reaching out to a friend (native speaker), I got a few pointers:

- Tame has a lot of possible translations, but in this case, ”飼いならす” or “おとなしい” sounds better.
- The に particle after ため can be omitted, making it a bit more conversational (like with a lot of particle uses).

<br />

So what I ended up with was the following:

<div className={'text-lg text-center'}>
  <Furigana
    characters={[
      { kanji: '居', kana: 'い' },
      { kanji: '酒', kana: 'ざか' },
      { kanji: '屋', kana: 'や' },
    ]}
  />
  のおとなしいサルを
  <Furigana characters={[{ kanji: '盗', kana: 'ぬす' }]} />
  んだため、
  <Furigana characters={[{ kanji: '怒', kana: 'おこ' }]} />
  られました。
</div>

<br />

## **Take aways:**

Apart from the new vocabulary itself, what I learned was the concept of ため:

ため have other possible uses, but in this case the concept was that of a cause having an effect.

Both cause and effect can be either a noun or an action (verb).

Given the structure `cause` (の)ため(に/の) `effect`, the following rules applies:

<br />

- _Noun_ のための _noun_
- _Noun_ のため(に) _verb_
- _Verb_ ための _noun_
- _Verb_ ため(に) _verb_
